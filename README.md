# 关于本项目
本项目为dart语言编写的，主要用于学习使用。适合初学者学习。
大部份的教程和实例代码都放在了项目的dart文件,并有详细的编号。
本项目将在2025.6之前,每天持续更新。
如果你感兴趣,欢迎点击右上角star。


Dart 是一种现代化、面向对象的编程语言，广泛用于构建移动、桌面、服务器和 Web 应用程序（尤其是 Flutter 框架）。
本项目旨在提供一个全面的 Dart 学习教程
